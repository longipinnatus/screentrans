一款支持自定义大模型 API 接口的 Android 屏幕 OCR 实时翻译工具。

你还可以根据文字区域大小、文字内容的正则表达式来忽略一些识别到的文本。

内置的OCR模型只能识别简体中文、繁体中文、英语、日语。但你也可以加载自定义的OCR模型以支持其他语言。


## 如何使用

默认是选择DeepSeek API，只需要在设置里输入API Key就可以翻译了。

在区域选择模式：单击悬浮球是框选翻译；双击可以进行更方便的垂直范围框选。

## 权限要求

必须开启的权限：

* 悬浮窗权限：覆盖原文翻译所必需
* 屏幕录制权限：用于截取屏幕图片进行OCR识别

推荐开启的权限：

* 通知权限：开启后可以稳定地通过后台Toast通知；
* 写入剪贴版权限：在一些定制系统里，需要设置为“始终允许”；
* 后台弹出界面权限：由于熄屏后系统会自动收回录屏权限，启用后可以方便地重新请求录屏权限；
